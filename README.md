# jqnav LazyVim

## Installation

### 1. Clone this repo

```bash
git clone https://github.com/juanQNav/jqnav-lazy-vim.git
```

### 2. Make a backup for your current Nvim configuration

```bash
cp ~/.config/nvim/ ~/.config/nvim.bak/
```

### 3. Set `jqnav-lazy-vim` as your nvim configuration

```bash
mv jqnav-lazy-vim ~/.config/nvim/
```
